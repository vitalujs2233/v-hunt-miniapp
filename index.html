---------------------------------------------------------------- ================================================== 2. ЧТО И КУДА ВСТАВЛЯТЬ ================================================== ШАГ 1. СОЗДАЙ НОВЫЙ РЕПОЗИТОРИЙ -------------------------------- Название можно сделать так: V-HUNT ВНУТРИ РЕПО СОЗДАЙ: - index.html И вставь в него код выше. ШАГ 2. ЧТО МЕНЯТЬ ВНУТРИ HTML -------------------------------- 1) Название проекта: Найди строку:
V-HUNT
Если хочешь другое имя — меняй только этот текст. 2) Подзаголовок: Найди строку:
Neon Arbitrage Terminal
Можешь поменять на свой вариант. 3) Ссылка на бота: Найди строку:
https://t.me/YOUR_BOT_USERNAME?startapp=ref_987654
СЮДА ПОТОМ ВСТАВИШЬ: https://t.me/ИМЯ_ТВОЕГО_БОТА?startapp=ref_987654 ПРИМЕР: https://t.me/vhunt_trade_bot?startapp=ref_987654 4) Кнопка Connect Wallet: Сейчас это заглушка. Позже сюда подключается TON Connect. ШАГ 3. ЧТО НУЖНО ВЗЯТЬ В TELEGRAM -------------------------------- Тебе нужен бот. Что делать: 1. Открой Telegram 2. Найди @BotFather 3. Напиши команду /newbot 4. Придумай имя бота 5. Придумай username бота 6. Получи BOT TOKEN СОХРАНИ: - BOT TOKEN - USERNAME БОТА BOT TOKEN нужен для backend. USERNAME бота нужен для ссылки: https://t.me/USERNAME_БОТА?startapp=... ШАГ 4. ЧТО НУЖНО ВЗЯТЬ ДЛЯ MINI APP -------------------------------- Тебе нужен публичный HTTPS-адрес сайта. Варианты: - Vercel - Netlify - GitHub Pages (для простого HTML) Лучше всего для старта: Vercel После публикации ты получишь ссылку типа: https://v-hunt.vercel.app ЭТУ ССЫЛКУ ты потом вставишь в BotFather как адрес Mini App. ШАГ 5. КАК ПОДКЛЮЧИТЬ MINI APP К TELEGRAM -------------------------------- Когда у тебя уже есть ссылка на сайт: 1. Открой @BotFather 2. Выбери своего бота 3. Настрой Mini App / Web App 4. Вставь ссылку на свой сайт Например: https://v-hunt.vercel.app После этого бот сможет открывать твой Mini App. ШАГ 6. ЧТО НУЖНО ДЛЯ TON CONNECT -------------------------------- Позже тебе понадобится ещё один файл: tonconnect-manifest.json Его нужно будет положить рядом с index.html. Пример содержимого: { "url": "https://ТВОЙ-ДОМЕН.vercel.app", "name": "V-HUNT", "iconUrl": "https://ТВОЙ-ДОМЕН.vercel.app/icon.png" } ГДЕ ЧТО БРАТЬ: - url = адрес твоего сайта - name = название проекта - iconUrl = ссылка на иконку проекта ШАГ 7. ЧТО НУЖНО ДЛЯ BACKEND -------------------------------- Для backend тебе потом понадобится: - BOT TOKEN из BotFather - DATABASE_URL из PostgreSQL - FRONTEND_URL = адрес твоего Mini App Пример .env: PORT=3000 BOT_TOKEN=СЮДА_ВСТАВЛЯЕШЬ_ТОКЕН_БОТА DATABASE_URL=СЮДА_ВСТАВЛЯЕШЬ_АДРЕС_БАЗЫ FRONTEND_URL=https://v-hunt.vercel.app ШАГ 8. ЧТО СЕЙЧАС МОЖНО ЗАГРУЗИТЬ СРАЗУ -------------------------------- Сейчас ты уже можешь: 1. создать новый репозиторий 2. загрузить туда index.html 3. открыть его как новый проект 4. опубликовать frontend 5. получить ссылку 6. подключить ссылку к боту ШАГ 9. ЧТО БУДЕТ ПОСЛЕ ЭТОГО -------------------------------- После того как HTML будет загружен и сайт опубликован, следующий этап: 1. сделать backend 2. сделать auth через Telegram initData 3. подключить TON Connect 4. подключить scanner API 5. сделать execute-логику ШАГ 10. ЧТО ТЕБЕ НУЖНО ЗАМЕНИТЬ ОБЯЗАТЕЛЬНО -------------------------------- ОБЯЗАТЕЛЬНО ПОТОМ ЗАМЕНИ: 1. YOUR_BOT_USERNAME 2. demo wallet address 3. demo referral link 4. demo numbers на карточках ШАГ 11. САМАЯ ПРОСТАЯ СХЕМА ДЛЯ ТЕБЯ -------------------------------- 1. Создаёшь репо 2. Загружаешь index.html 3. Публикуешь сайт 4. Создаёшь бота 5. Подключаешь сайт к боту 6. Тестируешь открытие внутри Telegram 7. Потом уже делаешь backend и wallet ШАГ 12. ЧТО Я УЖЕ ПОДГОТОВИЛ ДЛЯ ТЕБЯ -------------------------------- В этом файле у тебя уже есть: - готовый дизайн - готовый HTML - вкладки Scanner / Referrals / Wallet / Settings - кнопки - Telegram WebApp SDK - пометки что и куда вставлять
